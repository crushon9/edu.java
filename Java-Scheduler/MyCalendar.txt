3. 기능
ㄱ.메인기능
-일정등록db변경 : 날짜블럭클릭->오른쪽상세일정창에 목록출력->상단등록버튼클릭->새창1_일정입력창(시간,텍스트,글자배경색선택버튼클릭)->저장완료버튼->(db전송,일정리프레쉬)
-일정수정db변경 : 날짜블럭클릭->오른쪽상세일정창에 목록출력->해당일정행클릭->새창1_기존데이터를불러온상태->저장완료버튼->(db전송,일정리프레쉬)
-일정삭제db변경 : 날짜블럭클릭->오른쪽상세일정창에 목록출력->해당일정행클릭->새창1_기존데이터를불러온상태->삭제완료버튼->(db전송,일정리프레쉬)
-일정완료db변경 : 날짜블럭클릭->오른쪽상세일정창에 목록출력->해당일정행클릭->일정완료체크박스->완료여부true,옆에달력리프레시메소드(완료여부가false인것만담아서)호출)
-일정검색 : 전체일정중해당문자열검색/ 문자열입력 -> 검색버튼클릭 -> (db가져오기, jtable출력)
-일정출력리프레쉬 : db가져와서 orderby시간으로테이블정렬 j테이블출력/상단라벨값"날짜" /"검색문자열 : ㅇㅇㅇ"
-달력리프레쉬() : 옆에달력은일정중완료여부가flase인것만담아서출력

ㄴ.Imple기능
-리턴값 db등록전송 : excuteupdate, db가져와리턴()
-리턴값 db수정전송 : excuteupdate, db가져와리턴()
-리턴값 db삭제전송 : excuteupdate, db가져와리턴()
-리턴값 db완료값전송 : excuteupdate, db가져와리턴()
-리턴값 db가져와리턴(매개변수 날짜) : excutequary, 일정출력리프레쉬(매개변수 날짜)
-리턴값 db가져와리턴(매개변수 검색문자열) : excutequary, 일정출력리프레쉬(매개변수 문자열)

4.데이터구조- DB테이블
>테이블이름 : MEMBER
컬럼1 USER 사용자 varchar2 -(PK)
컬럼2 PASSWORD 비밀번호 varchar2 -notnull

>테이블이름 : SCHEDULE
컬럼1 SEQ - (PK) - 자동순번매겨짐
컬럼2 USER 사용자 varchar2 -로그인정보로 자동입력
컬럼3 DATE 날짜 date -notnull -일정삽입클릭된날짜자동입력
컬럼4 TIME 시간 number -notnull -사용자선택(콤보박스)
컬럼5 TEXT 일정내용 varchar2 -notnull -사용자입력
컬럼6 COLOR 배경색string배열인덱스 number -notnull -사용자선택
컬럼7 ISDONE 완료여부 String - 처음세팅값false -사용자체크박스클릭시true로전환

4.데이터구조- VO
>MemberVO
1String user 사용자
2String password 비밀번호

>ScheduleVO
1int seqNo 번호
2String user 사용자
3Date date 날짜
4int time 시간(0~24)
5String text 일정내용
6int colorIndex 배경색배열인덱스
7String isDone  완료여부
